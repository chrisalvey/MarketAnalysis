# Market Analysis

A market analysis dashboard hosted on GitHub Pages.

## Description

This project provides tools and visualizations for market analysis. It's designed to be lightweight, fast, and easily deployable via GitHub Pages.

## Features

- Interactive market data dashboard
- Data visualization and charting
- Responsive design for mobile and desktop
- Real-time analysis capabilities

## Getting Started

### Prerequisites

- A modern web browser
- Git installed on your system

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/chrisalvey/MarketAnalysis.git
   cd MarketAnalysis
   ```

2. Open `index.html` in your browser to view locally

### Deployment

This project is configured for GitHub Pages deployment.

#### Automatic Deployment (via Claude Code)

To deploy changes:
```bash
git add .
git commit -m "Your commit message"
git push origin main
```

GitHub Pages will automatically deploy from the main branch.

#### Manual Deployment

1. Push your changes to GitHub
2. Go to repository Settings > Pages
3. Select the main branch as the source
4. Your site will be published at: `https://chrisalvey.github.io/MarketAnalysis/`
